# 🤖 Тендерный агент ЕИС

Автоматический сборщик тендеров с zakupki.gov.ru на Python + Selenium.

## Структура проекта

```
tender_agent/
├── main.py                  # Точка входа
├── config.py                # Настройки (ключевые слова, фильтры, пути)
├── requirements.txt
│
├── scraper/
│   ├── browser.py           # Selenium с антибот-настройками
│   ├── eis_parser.py        # Парсинг страниц поиска ЕИС
│   └── downloader.py        # Скачивание документов
│
├── storage/
│   └── db.py                # SQLite: тендеры, документы, логи запусков
│
├── docs/                    # Скачанные документы (папка по номеру закупки)
└── logs/                    # Логи работы агента
```

## Установка

```bash
# 1. Создай виртуальное окружение
python -m venv .venv
source .venv/bin/activate      # Linux/macOS
.venv\Scripts\activate         # Windows

# 2. Установи зависимости (Chrome должен быть установлен)
pip install -r requirements.txt

# 3. Настрой ключевые слова в config.py
SEARCH_KEYWORDS = [
    "разработка программного обеспечения",
    "информационная система",
]
```

## Запуск

```bash
# Полный цикл: поиск + скачивание документов
python main.py

# Только поиск (без скачивания)
python main.py --no-download

# Только скачать документы (для уже найденных тендеров)
python main.py --only-download

# Конкретный запрос
python main.py --keyword "кибербезопасность"

# Ограничить кол-во тендеров на скачивание
python main.py --limit 20
```

## Что сохраняется

**SQLite база** (`storage/tenders.db`):
- `tenders` — метаданные закупок (номер, название, заказчик, цена, даты)
- `documents` — пути к скачанным файлам + тип документа
- `search_runs` — история запусков (когда, что искали, сколько нашли)

**Файлы** (`docs/<номер_закупки>/`):
- PDF, DOCX, XLSX и другие форматы документации
- Автоклассификация: ТЗ / проект_контракта / условия / прочее

## Следующий шаг — оркестрация с LLM

После накопления документов подключим локальную LLM (llama.cpp) для:
- Извлечения ключевых требований из ТЗ
- Оценки рисков
- Формирования дайджеста

## Важные нюансы ЕИС

- 44-ФЗ и 223-ФЗ имеют разные структуры URL
- FTP-выгрузки закрыты с января 2025
- SOAP API требует ЭЦП юр.лица
- Сайт иногда лагает — в агенте есть ретраи и случайные паузы
